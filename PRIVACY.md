# Privacy Policy

## Zeabur AI Hub Plugin

### Data Collection

This plugin does **not** collect any user personal data. It acts as a proxy to forward API requests to Zeabur's AI Hub services.

### Data Processing

- **API Requests**: User prompts and messages are sent directly to Zeabur's AI Hub API endpoints for processing.
- **API Keys**: Your Zeabur API key is stored locally within your Dify instance and is only used to authenticate requests.
- **No Logging**: This plugin does not log, store, or retain any user data or conversation content.

### Third-Party Services

This plugin uses Zeabur AI Hub API to access various AI models. Zeabur AI Hub provides enterprise-grade privacy with total isolation of the database from other AI vendors.

For information about how Zeabur handles data, please refer to [Zeabur's Privacy Policy](https://zeabur.com/docs/en-US/legal/privacy).
